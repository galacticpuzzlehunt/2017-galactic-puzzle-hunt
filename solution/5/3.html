Solution: Drive

Written by Anderson Wang

Each of the police reports describes a character or characters from a film, and each report can be paired up with the car that the given character(s) use throughout that film. This gives the exact path of each car in the street grid, and the facts that 1) none of the cars ever leave the grid, and 2) each intersection is used at least once, should help confirm the matching.

The fact that the cars are being chased by the police and the suggestion to “track down the cars” points at the next step: looking at license plates. For each car, the number of intersections it passes through is equal to the length of its license plate, which confirms that their license plates are relevant and also suggests writing them in the intersections of the street grid.

Writing the license plates in the intersections that the cars pass through gives the following:

There are 18 intersections in which 2 or more license plates overlap: in each of these intersections, there is exactly one letter and one or more digits. If the digits are summed up in these 18 intersections, the results are distinct numbers from 1 to 18, which means that we can pair each number 1-18 with a letter:

123456789101112131415161718
CROOKEDSTREETSSTAR

The star of the aptly-titled film Crooked Streets is ETHEL CLAYTON.

Author's Notes

A first draft of this puzzle had to be scrapped because I thought Rocky’s license plate in Rocky IV was S0THPAW instead of SOTHPAW (it didn’t help that the Rocky wiki even claims that it is S0THPAW!). It really looks like a 0, but while checking, I found out that digits on Pennsylvania license plates are larger than letters, which confirms that it is indeed SOTHPAW since they are all the same size. It had taken a long time to create a working grid, so I was considering leaving it as is and hoping that teams would realize that it was meant to be a 0, but in the end I obviously couldn’t leave a serious error in the puzzle.
